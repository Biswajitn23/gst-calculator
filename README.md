# GST Calculator

A simple GST Calculator built by Biswajit Nayak for calculating GST amount and final amount from a base amount and GST rate.

## Features

- Amount input
- GST rates: 5%, 12%, 18%, and 28%
- Real-time calculation after form submit
- GST amount and final amount output
- Digital Heroes button linking to https://digitalheroesco.com
